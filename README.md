# Shipping PDF Splitter

Tool to split shipping PDFs into individual pages organized by delivery number.

## Setup
1. Install Python dependencies: `pip install -r requirements.txt`
2. Update the PDF file path in `pdf_splitter.py`
3. Run: `python pdf_splitter.py`
